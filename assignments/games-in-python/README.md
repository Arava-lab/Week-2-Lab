# 🎮 Hangman Game Challenge

## 🎯 Objective

Build a playable Hangman game in Python that demonstrates use of strings, loops, conditionals, and user input handling. The goal is to practice program flow, state management, and basic I/O.

## 📝 Tasks

### 🛠️ Build the core game

#### Description
Implement the Hangman game that: selects a secret word, accepts letter guesses from the player, updates and displays progress, and ends with a win or lose message.

#### Requirements
Completed program should:

- Randomly select words from a predefined list.
- Accept letter guesses and show current progress (e.g., _ a _ _ m _ n).
- Track and display remaining incorrect guesses.
- End when the word is guessed or attempts are exhausted and display appropriate win/lose messages.

### 🛠️ (Optional) Add enhancements

#### Description
Extend the base game with one or more improvements from the ideas below.

#### Suggestions

- Add difficulty levels that change word list or allowed attempts.
- Implement hint functionality that reveals a letter.
- Keep a high-score or attempt log across sessions.
